# Security Policy

## Supported Versions

The following table indicates the versions of this project which
are currently being supported with security updates.

| Version | Supported          |
| ------- | ------------------ |
| 0.1.1   | :white_check_mark: |
| < 0.1.1 | :x:                |

## Reporting a Vulnerability

To report a vulnerability, please open a [GitHub Security Advisory](https://github.com/barcar/markdown-link-prefix/security/advisories/new) or contact the maintainers privately if you prefer.
